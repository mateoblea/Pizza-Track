# Proyecto Pizza-Track - Estructuras de Datos

## Objetivo
Este proyecto simula el sistema de gestión de pedidos de una pizzería utilizando dos pilas dinámicas lineales (Pila Principal y Pila Secundaria) creadas manualmente a partir de nodos y listas ligadas en Java. Permite gestionar el flujo de producción con las opciones de "Deshacer" (Undo) y "Rehacer" (Redo).

## Instrucciones de Ejecución
1. Abrir la carpeta del proyecto en VS Code.
2. Asegurarse de tener el JDK instalado (Eclipse Temurin recomendado).
3. Abrir el archivo `GestionPedidos.java`.
4. Ejecutar el proyecto presionando la tecla `F5`.
5. Interactuar con el menú numérico de la consola (1 para Registrar, 2 para Deshacer, 3 para Rehacer, 4 para Ver Actual, 0 para Salir).
